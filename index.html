Hola, soy Kevin Matsuda. Este es un sitio creado con el fin de que el docente pueda evaluar mis habilidades actuales sobre HTML, CSS, y Javascript. Su destino es académico.

La consigna es crear un sitio sencillo con 3 páginas HTML, algún estilo de portfolio, presentación, o copia de algún sitio de nuestro gusto. Aplicar estilos con CSS (a nuestro criterio), y funcionalidades o lógicas con Javascript. Si el proyecto tiene iconos o card por ejemplo no repetir el código HTML si no aplicar los métodos necesarios de JS para el renderizado de los iconos y o cards.
